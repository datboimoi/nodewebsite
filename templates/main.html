Node.js / Express Website

NodeJS

Node JS is a virtual runtime for Javascript. Unlike regular Javascript Node does not run on the browser but rather the back-end server of a website. Node is a asynchronous runtime, meaning it runs using a single thread instead of multiple threads which allow it to run faster than other back-end environements. Also being written in Javascript allows front-end developers to get in the fun of working with back-end data. :)

Express

Express is an open source framework downloadable with the help of npm (Node Package Manager). This allows for even quicker deployment of pages and data that is useful to clients and developers alike.
